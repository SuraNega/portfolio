FACEBOOK Full Stack Dev 2015

Experimental content creation feature that allows users to add to an existing story over the course of a day without spamming their friends. This project involved complex backend logic to manage story updates and frontend animations to create a seamless user experience.

The system was built with Ruby on Rails for the backend API and React.js for the frontend interface. We implemented real-time updates using ActionCable and optimized the database queries to handle high traffic volumes during peak hours.

HTML Ruby on Rails CSS JavaScript React PostgreSQL

Project Details

  • Client: Facebook Inc.
  • Team Size: 8 developers
  • Duration: 6 months
  • Challenges: Real-time updates, Performance optimization

Facebook 360 Full Stack Dev 2015

Exploring the future of media in Facebook's first Virtual Reality app: a place to discover and enjoy 360 photos and videos on Gear VR. This project pushed the boundaries of web-based VR experiences.

We developed custom WebGL components to render 360 content efficiently and built a content management system for artists to upload their work. The backend used Node.js with Express and MongoDB for flexible data storage.
